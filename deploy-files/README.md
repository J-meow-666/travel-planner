# AI旅游攻略平台

## 🌟 项目介绍

这是一个基于AI大模型的智能旅游攻略平台，支持用户输入目的地并生成个性化旅行计划，通过多轮对话实时调整行程安排。

## 🚀 功能特色

### 📍 智能行程规划
- 输入目的地即可生成完整的旅行计划
- 支持自定义天数、预算、旅行风格
- 基于兴趣爱好推荐活动安排
- 包含交通、住宿、餐饮、活动等全方位信息

### 💬 实时对话调整
- AI助手支持多轮对话交互
- 实时响应用户的调整需求
- 智能优化行程安排和预算分配
- 提供个性化建议和推荐

### 🎨 现代化界面
- 响应式设计，适配各种设备
- 优雅的动画和交互效果
- 直观的信息展示和操作
- 丰富的视觉反馈

### 🔒 安全可靠
- API密钥本地加密存储
- 支持多种AI大模型接入
- 完整的错误处理机制

## 📱 使用指南

### 1. 配置AI API
1. 打开网站后，在API配置区域输入您的API密钥
2. 确认API端点和模型名称
3. 点击"保存配置"按钮
4. 点击"测试连接"验证API可用性

### 2. 创建旅行计划
1. 输入目的地（如：北京、上海、杭州）
2. 设置旅行天数（1-14天）
3. 选择预算范围和旅行风格
4. 选择旅行人数和兴趣爱好
5. 点击"生成旅行计划"

### 3. AI对话调整
1. 在右侧聊天界面输入调整需求
2. AI会根据您的要求提供专业建议
3. 系统会自动更新行程安排
4. 可使用快捷建议按钮快速提问
